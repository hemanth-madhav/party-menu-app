## 🎯 Design Highlights

### Animations & Interactions
- **Smooth Transitions**: All elements use 0.3s cubic-bezier easing
- **Hover Effects**: 
  - Cards lift on hover
  - Buttons scale slightly
  - Color transitions smooth
- **Floating Effects**: 
  - Logo floats gently
  - Hero emoji bounces
- **GPU Accelerated**: Smooth 60fps performance

### Typography System
- **Display Font**: Plus Jakarta Sans (Bold headings)
- **Body Font**: Inter (Clean content)
- **Size Hierarchy**: Clear distinction between headings and body

### Responsive Design
- **Desktop** (1400px+): Full layout
- **Tablet** (768px-1199px): 2-column grid
- **Mobile** (480px-767px): Single column
- **Mobile XS** (<480px): Touch-optimized

### Shadow & Depth System
- **Small**: 1px shadow (subtle elements)
- **Medium**: 4-12px shadow (cards)
- **Large**: 12-32px shadow (floating elements)
- Increases on hover for visual feedback

## 🎨 Design System Variables

### Colors
```css
--primary: #FF6B35              /* Orange */
--primary-dark: #E85C2C         /* Darker Orange */
--secondary: #F37748            /* Coral */
--dark: #1F2937                 /* Navy */
--dark2: #111827                /* Darker Navy */
--veg: #10B981                  /* Green */
--nonveg: #EF4444               /* Red */
```

### Spacing
```css
--radius: 12px                  /* Border Radius */
--radius-lg: 16px               /* Large Radius */
--radius-xl: 20px               /* Extra Large Radius */
```

### Shadows
```css
--shadow: 0 4px 12px rgba(...)  /* Cards */
--shadow-lg: 0 12px 32px rgba(...)  /* Floating Elements */
```
